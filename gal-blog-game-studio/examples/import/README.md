# AI 导入示例

把任一剧本文件粘贴到 Studio 的“AI 创作”面板即可预览解析结果：

- `tagged.txt`：简单标签式 AI 剧本。
- `natural.txt`：中文自然描述。
- `scene.md`：Markdown 剧本。
- `scene.rpy`：Ren'Py-like 视图。
- `scene.webgal.txt`：WebGAL 原生脚本导入。
- `patch.json`：可逆 StoryPatch。
- `ai-tool-call.json`：语义工具调用；调用 API 时还需附上完整 `project`。

这些格式只是输入视图。应用后都变成同一份 Story IR。
