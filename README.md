## AI 使用规则

1. 开始任务前，先阅读本文件和目标主题文件夹中的 `README.md`。
2. ChatGPT、Claude、Grok 等 AI 均可读取、整理和修改所有主题；某个文件夹可以注明常用 AI，但不构成限制。
3. 默认只处理用户指定的主题文件夹，不随意改动其他主题。
4. 默认直接提交到 `main`，不创建分支或 Pull Request，除非用户明确要求。
5. 只归档有长期价值、需要继续使用或需要持续更新的内容。
6. 新增重要文件后，在该主题的 `README.md` 中补充文件索引。
7. 写入完成后，应说明修改了哪些文件，并提供对应 commit。
8. 这是公开仓库：不得保存密码、Token、私人身份信息或其他敏感资料。
