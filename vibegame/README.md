# VibeGame

## 项目是什么

把浅仪式的个人博客做成尽可能接近日本 PC Galgame 标题界面的 Web 体验，而不是普通网页叠加一张二次元背景。

目前优先完成爱丽丝主题，其他看板娘与主题以后逐步扩展。

## 已确认的核心方向

- 网站具有 Galgame 的启动感：首次进入先播放简短开场，点击或触摸可跳过并进入主菜单。
- 背景、角色、标题和菜单应融为一个完整场景，避免普通网页式左右硬分栏。
- 主菜单可使用 `START / LOAD / CONTINUE / EXTRA / OPTION / EXIT` 等经典项目，并映射为博客功能。
- `START` 进入简易 Web Galgame；`LOAD` 映射文章列表；`EXTRA` 可承载音乐与额外资源。
- 支持 BGM、日夜变化、天气特效，以及 PC、平板和手机的响应式适配。
- 页面切换应有自然过渡，避免白屏、组件闪烁或明显的网页跳转感。
- 标题暂定为「君が読む永遠」。

## 当前角色与视觉方向

第一位看板娘为爱丽丝：

- 金色长发、蓝色眼睛、小皇冠、深蓝与白色女仆裙。
- 整体偏 2008～2010 年日本 PC 美少女游戏画风。
- 偏好二维手绘、清晰线稿、适度赛璐珞上色。
- 避免现代手游 UI、3D 渲染、照片写实、网红背景和过度电影化光影。
- 额头红宝石是直接镶在额头上的，不能出现两根挂绳；头发只需稍微露出宝石。
- 爱丽丝称呼浅仪式为“主人”。

## 当前动态立绘方案

正在尝试让单张 PNG 产生简单的 2.5D 动态效果：

- 目标不是平面左右摆动，而是通过视觉形变产生轻微的 Z 轴前后摆动感。
- 已有编辑工具可以导出描述效果的 JSON 配置。
- 前端需要同时读取原始 PNG 与 JSON，并按照配置渲染波纹或摆动效果。
- 实现时应避免头发、身体等区域出现割裂感。
